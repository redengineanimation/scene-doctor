# Lite vs Pro

Scene Doctor ships as a free **Lite** build and a paid **Pro** build. The
boundary is **detect vs. fix + advanced checks + automation** — not
"which basic checks."

| | Lite | Pro |
|---|:---:|:---:|
| Core checks — detect, drill-down, health score | ✅ (5) | ✅ (5) |
| Apply-transforms fix | ✅ | ✅ |
| Advanced checks | — | ✅ (11 more) |
| One-click fixes (recalc, purge, merge, relink…) | — | ✅ |
| Missing-file folder search & relink | — | ✅ |
| Profiles (Game / 3D-Print / Render) | — | ✅ |
| Fix-All-Safe batch | — | ✅ |
| CSV report export | — | ✅ |
| Naming presets (Unity / Unreal / Godot) | — | ✅ |

**Lite** runs the five core checks in detect mode and shows exactly what
Pro unlocks — every gated control is visible with a *(Pro)* label.

**Pro** unlocks all sixteen checks, every one-click fix, profiles,
batch fixing, and report export.

!!! info "Honest, keyless licensing"
    Scene Doctor is GPL-licensed and keyless. The tier split is an honest
    product boundary, not DRM. The Lite build simply doesn't include the
    Pro code.
